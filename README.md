**# Carry-On**



Projeto desenvolvido no âmbito da Prova de Aptidão Profissional (PAP) do Curso Técnico de Gestão de Equipamentos Informáticos.



\## Descrição

O projeto Carry-On consiste no desenvolvimento de duas aplicações móveis para gestão de pedidos num bar e de um protótipo de robô de transporte de bebidas.



\## Componentes do projeto

\- Aplicação móvel para clientes

\- Aplicação móvel para funcionários

\- Protótipo de robô com Arduino Mega



\## Tecnologias utilizadas

\- .NET MAUI

\- C#

\- Firebase Realtime Database

\- Arduino Mega 2560 R3

\- RFID

\- Sensores ultrassónicos

\- Motores de passo



\## Estrutura do repositório

\- /apps – Código-fonte das aplicações móveis

\- /robot – Código-fonte e documentação do robô



\## Instruções

\### Aplicações

Abrir os projetos .NET MAUI no Visual Studio e compilar.



\### Robô

Abrir o ficheiro `.ino` na Arduino IDE e carregar para a placa Arduino Mega.
